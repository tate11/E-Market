Theme Clarico

Fully Responsive Odoo eCommerce Theme

Clarico is a Responsive, Clean, Modern & Sectioned Odoo eCommerce Theme. Crafted to be Pixel Perfect, it is built using Bootstrap Grid and can be suitable for any kind of online store category like Fashion, Electronics, Beauty, Health & Fitness, Jewelleries, Sports etc. It is a fully responsive theme and looks equally stunning on all kinds of screens and devices. It is built with various attractive features like 2 unique Home Page Layouts, 4 Category Page Layouts, 3 custom Mega Menu Layouts, 6 different Color Schemes, 15 Custom Designed Snippets etc., that would take months to build. Information blocks are neatly displayed to customers so they can easily find the information they need. Rest assured, Clarico will make your web store look stunningly beautiful.
